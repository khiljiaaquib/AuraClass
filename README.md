## 📌 About

AuraClass is an AI-powered classroom attendance system that automates attendance using facial recognition. It eliminates manual attendance, provides real-time reports, and offers separate dashboards for teachers and students.

---

## ✨ Features

### 👨‍🏫 Teacher Panel

- Secure Teacher Login
- Create Subjects
- Enroll Students
- Capture Classroom Photos
- AI Face Recognition Attendance
- Attendance History
- Subject-wise Attendance Records
- Student Management
- Real-time Dashboard

### 👨‍🎓 Student Panel

- Secure Login
- View Attendance Percentage
- View Enrolled Subjects
- Attendance History
- Profile Management

### 🤖 AI Features

- Face Detection
- Face Recognition
- Automatic Attendance Marking
- Multi-face Detection
- Fast Attendance Processing

